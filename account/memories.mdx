---
title: "Memories: personalize your Magica AI responses"
description: "Store details about yourself so Magica remembers your preferences, skills, and context across every conversation without repeating yourself."
---

The Memories feature lets you save details about yourself — your job role, writing style, recurring projects, or anything else useful — so the AI can give more relevant, personalized responses without you having to re-explain your context every time.

## Add a memory

<Steps>
  <Step title="Open Workspace Settings">
    Click the gear icon beside **Recent** in the left sidebar.
  </Step>
  <Step title="Click + Add Memory">
    Select **+ Add Memory** in the Memories section.
  </Step>
  <Step title="Enter your note">
    Type or paste the detail you want to store. Each memory can be up to 2,500 characters.
  </Step>
  <Step title="Save">
    Click **Add Memory** to save it. The AI will apply this context to future conversations.
  </Step>
</Steps>

<Tip>
  Useful things to store as memories: your job title and industry, preferred writing tone, programming languages you work in, or the context of a long-running project.
</Tip>

## Import memories from ChatGPT

If you have existing memories in ChatGPT, you can transfer them to Magica:

1. In ChatGPT, go to **Settings → Personalization → Manage Memories**.
2. Copy the memories you want to bring over.
3. Paste them into a new memory in Magica's Workspace Settings.

## Managing a full memory

If you reach the memory limit, Magica will notify you. To free up space, go to **Workspace Settings → Memories** and delete any individual memories you no longer need.
