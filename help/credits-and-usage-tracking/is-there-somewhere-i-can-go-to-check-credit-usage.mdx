Yes! You can view both total and individual tool usage at [https://magica.com/usage](https://magica.com/usage).

**Detailed Explanation**

* The usage dashboard shows a breakdown by tool or feature, letting you see which actions consume the most credits.
* You’ll also see toast notifications when you use each feature.

**Troubleshooting & Tips**

* If you notice inconsistencies, log out and back in to refresh the data.
* Contact support if the usage stats still appear incorrect.
