Both **Magica Auto** and **Magica Max** have access to all leading AI models internally.

The agent intelligently decides which model to use for each specific task or query automatically, so you do not need to manually choose a model for every request.

The difference is:

* **Magica Auto** is designed to be a more budget-friendly option for general usage.
* **Magica Max** is built for advanced or heavier usage, with higher limits and more intensive reasoning capabilities.

In short: both can use top models, but Magica Max is better suited for demanding workflows.
