Each AI tool page on Magica has a “View History” button to see your previously generated results for that specific tool.

* There is currently **no centralized history page**. Instead, go to each tool (e.g., AI Image Generator, AI Video Generator) to see its unique history.
* This method helps keep relevant generations grouped by the tool used.

**Troubleshooting & Tips**

* If you don’t see your past creations, ensure you’re logged into the same account you used to generate them.
* If the tool page doesn’t display a “View History” button, try refreshing or contact support.
