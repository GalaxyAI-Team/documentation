Yes, Magica chat models have real-time web access. You can enable the web search feature by clicking on the globe icon in the message input bar.

* Magica uses advanced AI to search the internet and gather insights in real time.
* This allows for up-to-date answers, unlike static models trained on older data.

**Troubleshooting & Tips**

* If you’re not seeing web-based results, ensure the globe icon is toggled on.
* For specialized tasks (e.g., web scraping or large queries), ensure you have enough credits.
