Steps:

1. Open **Workspace Settings**
2. Click **+ Add Memory**
3. Type or paste your note (up to 2,500 characters)
4. Click **Add Memory** to save

**Tip:** You can import memory from ChatGPT via **Settings → Personalization → Manage Memories**, then paste it into Magica.
