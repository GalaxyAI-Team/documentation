title: Can I know upfront how many credits a task will use?
-->
Credit usage can't be predicted upfront because it's based on the actual computational cost of each generation. This depends on several factors like the model you use, prompt length, output length, and quality settings.

That said:

* Track your detailed credit usage anytime here: [magica.com/usage](https://magica.com/usage)
* Failed generations are never charged, so you won't pay for errors
* You can see individual credits via toast notifications while using each tool
