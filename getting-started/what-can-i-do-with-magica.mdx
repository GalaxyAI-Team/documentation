Magica gives you several ways to use AI, depending on how you want to work:

- **Super Agent:** Get work done with an autonomous AI agent that has capable AI models, a computer, browser, and connectors for accessing your apps and services. [Open Super Agent](https://magica.com/app/chat)
- **Flow:** Build AI workflows visually. Chain models and tools together without code. [Open Flow](https://magica.com/app/flow)
- **Nodes:** Run AI models and tools directly when you want a specific capability. [Open Nodes](https://magica.com/app/nodes)
- **API / MCP:** Use Magica from your code or from an MCP-aware agent. [View API and MCP docs](https://magica.com/docs)

If you are not sure where to start, use **Super Agent** for general tasks, **Flow** for repeatable workflows, **Nodes** for direct model/tool runs, and **API / MCP** for developer or agent integrations.
